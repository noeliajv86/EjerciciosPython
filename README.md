# Ejercicios de Python



Este repositorio contiene mis prácticas de programación en Python como parte de mi proceso de aprendizaje. 

##Se advierte que la terminal de Visual estudio Code que se utilizó no es la que viene integrada. 
Debí modificarla a una externa por las limitaciones de mi versión de Windows que ya no posee soporte de Microsoft y no es compatible con la última versión de Visual Studio Code. 

Por ejemplo , no reconoce los "float" porque uso la terminal externa de Windows o CMD y los colores podrian diferir. Las extensiones no son las mismas. 


## 🚀 Contenido del proyecto
* **Ejercicios básicos:** Variables, tipos de datos y operaciones simples.
* **Estructuras de control:** Condicionales (`if/else`) y bucles (`for/while`).
* **Prácticas recomendadas:** Uso de buenas prácticas guiadas por IA.
